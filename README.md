# 双UPS智能监控与充放电控制系统

> **版本：** v1.0（工程发布版）  
> **硬件平台：** Seeed Studio XIAO SAMD21 + 微雪 UPS Module 3S × 2  
> **核心特性：** 双路独立I2C、继电器安全仲裁、心跳存活检测、LED异常定位、工业级错误恢复

---

## 📌 项目简介

本项目是一个**高可靠性双电池UPS（不间断电源）监控与充放电控制器**固件，运行于 XIAO SAMD21 单片机，专为树莓派等关键设备设计。

系统通过两颗 INA219 实时监测双电池组的电压、电流与功率，配合继电器实现**自动充放电管理**；并通过串口心跳机制与树莓派联动，一旦主机失联立即切断充电，防止过充风险。

**最大亮点：**  
✅ 采用 **双色LED轮询 + 红色快闪** 机制，无需屏幕即可**通过肉眼直接定位故障电池**。

## 🔧 硬件配置

### 核心硬件
- **主控：** Seeed Studio XIAO SAMD21
- **电源管理：** Waveshare UPS Module 3S × 2（内置 INA219）
- **切换器件：** 继电器模块 × 2（控制充电回路）

### 硬件连接表

| XIAO SAMD21 | 外设 |
|-----------|------|
| D0 | 继电器 A（电池A充电控制） |
| D1 | 继电器 B（电池B充电控制） |
| D2 (SDA) | UPS A I2C SDA |
| D3 (SCL) | UPS A I2C SCL |
| D4 (SDA) | UPS B I2C SDA |
| D5 (SCL) | UPS B I2C SCL |
| D6 (TX) | 树莓派 RX（Serial1） |
| D7 (RX) | 树莓派 TX（Serial1） |
| D8 | 双色LED 绿色 |
| D10 | 双色LED 红色 |
| USB | 调试串口（Serial） |

## ✨ 核心功能

### 🔋 电池管理
- ✅ 双路 **独立 I2C 总线**，避免地址冲突
- ✅ INA219 实时监测电压 / 电流 / 功率
- ✅ 基于电压估算 SOC（State of Charge）
- ✅ 自动充放电状态机（Idle / Charging / Discharging / Full / Error）

### ⚡ 充电控制（安全优先）
- ✅ **互斥充电仲裁**：同一时刻仅允许一块电池充电
- ✅ SOC 滞回控制（<20% 开启，>95% 停止）
- ✅ 继电器防抖（2秒）
- ✅ 充电切换延迟（1秒），防止频繁动作

### 💓 主机心跳保护（防过充）
- ✅ 树莓派通过串口发送 `CHG:1/0` 心跳
- ✅ 心跳超时（3秒）自动禁止充电
- ✅ 主机宕机 → 立即切断充电回路

### 💡 LED 可视化诊断（核心特色）
| LED 行为 | 含义 |
|--------|------|
| **绿色呼吸（2s）** | 电池A 正常 |
| **黄色呼吸（2s）** | 电池B 正常 |
| **常亮** | 正在充电 |
| **短呼吸（1s）** | 正在放电 |
| 🔴 **红色快闪（4Hz）** | **异常（ERROR 或 SOC<20%）** |

👉 **每4秒切换显示 A / B，通过观察“何时出现红灯”即可定位故障电池。**

### 🛡️ 工业级错误恢复
- ✅ I2C 通信异常自动进入 ERROR 状态
- ✅ **连续5次有效采样 + 2秒冷却期** 才允许恢复
- ✅ 错误日志去重，防止串口刷屏
- ✅ 低电量告警支持 **1小时重复上报窗口**

## 🖥️ 树莓派端协议

### 心跳协议（必需）
树莓派需每秒向 `/dev/ttyS0`（或对应串口）发送：
```text
CHG:1\n
```
或
```text
CHG:0\n
```

| 指令 | 含义 |
|----|----|
| `CHG:1` | 允许充电 |
| `CHG:0` | 禁止充电 |

### 控制器返回日志
| 日志 | 含义 |
|----|----|
| `HB:ALLOW` | 心跳允许 |
| `HB:DENY` | 心跳禁止 |
| `HB:TIMEOUT` | 心跳超时 |
| `ERR:A,INVALID_VOLTAGE` | 电池A异常 |
| `ERR:B,LOW_SOC` | 电池B低电量 |
| `RECOVER:A` | 电池A恢复 |

## 📊 状态输出格式

串口每秒输出一次紧凑状态：
```text
A,7.82,-120,78.5,DIS,OFF B,8.10,350,92.0,CHG,ON,ACT_B
```

字段说明：
```
电池名,电压(V),电流(mA),SOC(%),状态,继电器状态  电池B... ,当前充电目标
```

## ⚠️ 重要安全提示

1. ⚠️ **UPS Module 3S 的 INA219 I2C 地址固定为 0x40，必须使用独立 I2C 总线**
2. ⚠️ 继电器仅控制充电回路，**切勿用于切断主供电**
3. ⚠️ SOC 基于电压估算，仅供参考，不建议用于高精度场景
4. ⚠️ 首次上电建议观察 LED 与串口日志，确认双路 INA219 初始化成功
